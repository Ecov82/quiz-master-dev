## Como Executar a Aplicação (Passo a Passo)

Siga as instruções abaixo para configurar e rodar o projeto localmente em sua máquina.

### 1. Pré-requisitos
Antes de começar, certifique-se de ter instalado em sua máquina:
* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (ou a tecnologia principal utilizada na aula)

### 2. Clonar o Repositório
Abra o terminal do seu computador e execute o comando abaixo para baixar este projeto:
```bash
git clone [https://github.com/Ecov82/quiz-master-dev.git](https://github.com/Ecov82/quiz-master-dev.git)

```

### 3. Acessar a Pasta do Projeto

Navegue até o diretório criado:

```bash
cd quiz-master-dev

```

### 4. Instalar as Dependências

Instale todos os pacotes necessários para o funcionamento da aplicação:

```bash
npm install

```

### 5. Iniciar o Servidor/Aplicação

Agora, basta iniciar o projeto com o comando:

```bash
npm start

```

Abra o seu navegador e acesse o endereço indicado no terminal (geralmente `http://localhost:3000`).
