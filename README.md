# Quiz Master 

O **Quiz Master** é uma aplicação interativa de perguntas e respostas desenvolvida para testar conhecimentos de forma dinâmica e divertida.

---

##  Como Executar a Aplicação (Passo a Passo)

Siga as instruções abaixo para configurar e rodar o projeto localmente em sua máquina.

### 1. Pré-requisitos
Antes de começar, certifique-se de ter instalado em sua máquina:
* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (ou a tecnologia principal utilizada no seu projeto, ex: Python, Java, etc.)

### 2. Clonar o Repositório
Abra o terminal do seu computador e execute o comando abaixo para clonar o projeto:
```bash
git clone [https://github.com/Ecov82/quiz-master.git](https://github.com/Ecov82/quiz-master.git)

```

### 3. Acessar a Pasta do Projeto

Navegue até o diretório criado:

```bash
cd quiz-master

```

### 4. Instalar as Dependências

Instale todos os pacotes necessários para o funcionamento da aplicação:

```bash
npm install
# Caso use outra tecnologia, substitua pelo comando correspondente (ex: pip install -r requirements.txt)

```

### 5. Iniciar o Servidor/Aplicação

Agora, basta iniciar o projeto com o comando:

```bash
npm start
# Ou o comando correspondente para rodar o ambiente de desenvolvimento

```

Abra o seu navegador e acesse o endereço indicado no terminal (geralmente `http://localhost:3000`).

---

##  Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (ou React / Vue, se aplicável)
