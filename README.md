# Quiz Master (Versão do Aluno)

O **Quiz Master** é uma aplicação interativa de perguntas e respostas. Este repositório contém a estrutura base do projeto para que os alunos possam desenvolver, praticar e implementar novas funcionalidades.

---

## Objetivo da Atividade

O sistema base do **Quiz Master** já está estruturado, mas o nosso banco de conhecimentos precisa crescer. Sua missão como desenvolvedor(a) é expandir o projeto!

**Suas tarefas:**
1. **Criar Novas Categorias:** Adicione pelo menos 3 novas categorias temáticas ao quiz (ex: Ciência, Tecnologia, História do Brasil, Cultura Pop).
2. **Alimentar o Banco de Perguntas:** Crie no mínimo 5 perguntas para cada uma das novas categorias. Certifique-se de formatar corretamente as opções e definir a resposta exata.
3. **Validar o Fluxo:** Teste a aplicação e garanta que o menu principal esteja listando as novas categorias automaticamente e que a pontuação esteja sendo calculada corretamente ao final das novas rodadas.

---

## Como Executar a Aplicação (Passo a Passo)

Siga as instruções abaixo para configurar e rodar o projeto localmente em sua máquina.

### 1. Pré-requisitos
Antes de começar, certifique-se de ter instalado em sua máquina:
* [Git](https://git-scm.com/)
* [Python](https://www.python.org/) (versão 3.8 ou superior recomendada)

### 2. Clonar o Repositório
Abra o terminal do seu computador e execute o comando abaixo para baixar este projeto:
```bash
git clone [https://github.com/Ecov82/quiz-master-dev.git](https://github.com/Ecov82/quiz-master-dev.git)

```

### 3. Acessar a Pasta do Projeto

Navegue até o diretório criado:

```bash
cd quiz-master-dev

```

### 4. Instalar as Dependências

*(Se você costuma utilizar ambientes virtuais como `venv`, crie e ative o seu ambiente antes deste passo).*

Instale os pacotes necessários para o funcionamento da aplicação:

```bash
pip install -r requirements.txt

```

*(Nota: Se o projeto utilizar `pyproject.toml` para gerenciar dependências, você pode rodar o comando `pip install .` no lugar)*

### 5. Iniciar a Aplicação

Agora, basta iniciar o projeto executando o arquivo principal:

```bash
python main.py

```
